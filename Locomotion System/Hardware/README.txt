NOTA
 - Os motores utilizados na simulação do Proteus são diferentes dos do circuito real, são apenas representação. Por isso não é possível ver na simulação os motores a rodar corretamente, mas na prática está tudo a funcionar. Pode ser provado se no Proteus ligarmos um osciloscópio nas saídas.

NOTE
 - 
The motors used in the Proteus simulation are different from those of the real circuit, they are only representation. So it is not possible to see the engines running correctly in the simulation, but in practice everything is working. It can be proved if in Proteus we connect an oscilloscope to the outputs.
